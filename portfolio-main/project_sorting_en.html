Sorting Plant with PLC

Academic Project

Project Description

This project involved the complete design, conception, and programming of an automated sorting plant. The primary goal was to sort objects based on specific criteria (e.g., color or material) using a variety of sensors. The logic and control of the entire system, including conveyor belts, sensors, and pneumatic actuators, were implemented using PLC technology. This provided hands-on experience in industrial automation and control systems.

Technologies Used:

  • PLC Programming (TIA Portal)
  • Sensor Integration (Inductive, Capacitive, Optical)
  • Actuator Control (Pneumatics)
  • System Design & Simulation
