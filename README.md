## 📋 Über das Projekt

Diese Portfolio-Webseite wurde entwickelt, um meine beruflichen Erfahrungen, technischen Fähigkeiten und akademischen Projekte zu präsentieren. Die Seite ist vollständig responsiv und in drei Sprachen verfügbar.

### ✨ Features

- **Mehrsprachig:** Vollständige Unterstützung für Deutsch, Englisch und Arabisch
- **Responsives Design:** Optimiert für Desktop, Tablet und Mobile
- **Interaktive Projektseiten:** Detaillierte Beschreibungen mit KI-generierten Bildern
- **Moderne UI:** Sauberes Design mit Tailwind CSS
- **Schnelle Ladezeiten:** Optimiert für beste Performance

## 🚀 Lokale Entwicklung

Falls du das Projekt lokal ausführen möchtest:

1. **Repository klonen:**
   ```bash
   git clone https://github.com/moaloush/portfolio.git
   cd portfolio
   ```

2. **Lokalen Server starten:**
   ```bash
   # Mit Python 3
   python -m http.server 8000

   # Mit Node.js (falls installiert)
   npx serve .
   ```

3. **Im Browser öffnen:**
   ```
   http://localhost:8000/portfolio.html
   ```
